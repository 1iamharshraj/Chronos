# Chronos

Chronos is an elegant, customizable fullscreen clock web application built with [Next.js](https://nextjs.org/) and modern web technologies.

## ✨ Features

- **Minimalist Clock Display**: Clean, distraction-free interface.
- **Customizable Format**: Toggle between 12-hour and 24-hour time formats.
- **Date & Location**: Option to display the current date and location information.
- **Fullscreen Mode**: Immerse yourself completely with fullscreen support. 
- **Persistent Settings**: Your preferences are saved automatically in your browser's local storage.
- **Keyboard Shortcuts**: Power-user friendly with simple keybindings.
- **Responsive Design**: Looks great on screens of all sizes.

## ⌨️ Keyboard Shortcuts

- `F`: Toggle Fullscreen mode
- `S`: Open Settings panel
- `Esc`: Close Settings or exit Fullscreen mode

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🛠️ Getting Started

First, ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps to run Chronos locally:

### 1. Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

### 2. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### 3. Open in your browser

Open [http://localhost:3000](http://localhost:3000) with your browser to see the live clock.
